-----------------------------Задание--------------------------------

Установить 1С-битрикс на сервер

Параметры для SSH подключения:
IP сервера: ******
Имя пользователя: *****
Пароль: *****

Создать раздел Калькулятор ( /calc/ )
В разделе калькулятор разместить компонент, со следующими условиями:
Вывести на странице форму, содержащую поля ввода для следующих параметров:
длина комнаты,
ширина комнаты,
длина доски ламината
ширина доски ламината

После ввода значении и нажатия кнопки «Рассчитать» форма должна отправить запрос к php скрипту компонента, который выполняет расчет минимально необходимого количество ламината для укладки в комнате с заданными параметрами, и после получения ответа отобразить результат на странице с формой.
